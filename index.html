Talor's Name Website

My name is pronounced "Tah-lore"

My name is hebrew and originates from the hebrew words Tal (dew) and Ohr (light)

My name means morning dew

To create this webpage, I linked my HTML file to Google Fonts so that I could use the Corgette font, changed the font color and size, added a GIF, and created a black background with white sparkles. The background required creating an additional CSS file to layer a PNG file on top of the black background.
